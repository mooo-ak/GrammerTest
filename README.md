# ToyProject_GrammerTest
스프링 MVC + JSP _ 맞춤법 테스트 사이트
<br>

## 🖥️ 프로젝트 소개
헷갈릴 수 있는 한글 맞춤법에 대해 간단히 테스트하고, 결과를 공유하는 사이트입니다.
<br>

### ⚙️ 개발 환경
- `Java SE 17`
- `JDK 17.0.10`
- **IDE** : Eclipse 4.23.0
- **Framework** : SpringFramework(5.3.20)
- **Database** : Oracle DB(21c)
- **ORM** : Mybatis
<br>
## 📌 주요 기능
#### 맞춤법 테스트 
- 질문 및 답안 선택
- JSON 데이터 형식
- Ajax를 사용하여 테스트 결과 반환
#### 결과 공유하기 
- 카카오톡 API 연동
- 결과 공유하고 추가 참여 유도
